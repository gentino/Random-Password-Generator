# Random Password Generator

## Description
This is a simple Python program that generates random passwords based on the user's input. The user specifies how many passwords to generate and the number of characters for each password. The generated passwords contain a mix of letters, numbers, and special characters.

## Features
- Generates one or more random passwords.
- Allows the user to choose the password length.
- Uses uppercase letters, lowercase letters, digits, and special characters.
- Validates user input.
- Limits invalid input attempts to three.

## Requirements
- Python 3.10 and above

## Modules Used
- `random`
- `string`

## How to Run
1. Save the program as a `.py` file.
2. Open a terminal or command prompt.
3. Run the program:
   ```bash
   python pasword_generator.py
   ```
4. Enter:
   - The number of passwords to generate.
   - The number of characters for each password.
5. The generated passwords will be displayed on the screen.
